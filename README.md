# ❤️ Heartline404

**Heartline404** is a story-driven Progressive Web App (PWA) that feels like watching a Netflix teen drama entirely through a realistic smartphone interface.

## 🎯 Vision

Players don't solve mysteries or make constant choices.

Instead, they experience a messy, emotional world through:
- 💬 Chats
- 📸 Social media
- 📞 Calls
- 🖼️ Photos
- 🎧 Voice notes
- 📅 Calendar events
- 📝 Notes
- 🔔 Notifications

The goal is to make players feel like they accidentally became part of someone else's life.

## 🛠️ Tech Stack

- React
- Next.js
- Tailwind CSS
- Firebase
- Progressive Web App (PWA)
